### Quel est le but de ce projet ?

Le but de ce projet est de nous faire travailler sur le JavaScript et de lier le front (affichage client) au back (l'arrière plan du site). Puisque le sujet est un exercice de formation je ne pourrais pa vraiment détailler les étapes de mon travail mais je vais tout de meme présenter celui-ci de la meilleure manière possible.

### La mise en place :
